10. Concatenation of Character and String

public class CharToStr {
    public static void main(String[] args) {
        String str1 = "Java";
        char str2[] = {'J', 'a', 'v', 'a'};
        String str3 = null;
        for (char c : str2) {
            str3 = str3 + c;
        }
        if (str1.equals(str3)) {
            System.out.println("Successful");
        } else {
            System.out.println("Unsuccessful");
        }
    }
}

What is the result?

A. Successful

B. Unsuccessful

C. Compilation fails

D. An exception is thrown at runtime

  • str3 一開始為 null，接著直接使用「+」運算子與字元做字串連接（Java 允許此種作法）， 但是 String str3 = null; 在字串連接時會自動被改為長度為 4 的「null」字串， 輸出結果為「nullJava」，因此結果為 Unsuccessful。

  • 「+」運算子的運算元必須要至少有一個是字串 String 或者物件 Object，才會被當成字串連結，不然就是加法

            // System.out.println(null + 'c'); // bad operand types (不知型態 + 字元值99)
        String str = null;
        System.out.println(str + "c");                    // nullc
        System.out.println(null + "c");                   // nullc
        System.out.println(str + 'c');                    // nullc
        System.out.println('n' + 'u' + 'l' + 'l' + 'c');  // 542
        System.out.println((String) null + 'c');          // nullc

18. How to sout joe:true:100.0 ?

public class Employee {

    String name;
    boolean contract;
    double salary;

    Employee() {
        // line n1
    }

    public String toString() {
        return name + ":" + contract + ":" + salary;
    }

    public static void main(String[] args) {
        Employee e = new Employee();
        // line n2
        System.out.println(employee);
    }
}

Which two modifications, when made independently, enable the code to print joe:true:100.0?

A. Replace line n2 with:

e.name = "Joe";
e.contract = true;
e.salary = 100;

B. Replace line n2 with:

this.name = "Joe";
this.contract = true;
this.salary = 100;

C. Replace line n1 with:

this.name = new String("Joe");
this.contract = new Boolean(true);
this.salary = new Double(100);

D. Replace line n1 with:

name = "Joe";
contract = TRUE;
salary = 100.0f;

this("Joe", true, 100);

22. Permanent passing value

class Product {

    double price;
}

public class TestPrt {

    public void updatePrice(Product product, double price) {
        price = price * 2;
        product.price = product.price + price;
    }

    public static void main(String[] args) {

        Product prt = new Product();
        prt.price = 200;
        double newPrice = 100;

        TestPrt t = new TestPrt();
        t.updatePrice(prt, newPrice);
        System.out.println(prt.price + " : " + newPrice);
    }
}

What is the result?

A. 200.0 : 100.0

B. 400.0 : 200.0

C. 400.0 : 100.0

D. Compilation fails.

  • Java 永遠是 pass by value
    • line 20: 同時將基本型態的區域變數 newPrice 和物件參考變數 prt 當作參數傳給 updatePrice() 方法
      • newPrice 和 prt 的值都會被複製一份出來，prt的值為 Product 物件的參考 (類似記憶體位置)
      • updatePrice() 方法會將 newPrice 的值乘以 2，再加給 prt 物件的 price 變數
    • line 21:
      • 將 prt 物件的 price 變數的值和 newPrice變數的值輸出
      • prt 物件的 price 變數的值會是執行 updatePrice() 方法時改變的結果
