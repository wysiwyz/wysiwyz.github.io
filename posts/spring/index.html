Spring Architecture

Core Container

  1. Core - 提供框架最基本的部分
  2. Bean - 提供Bean工廠
  3. Context - 存取任何定義且被配置的物件
  4. SpEL - Spring express language

Data Access

  1. JDBC - 提供JDBC抽象層
  2. ORM - 提供Object-Relational Mapping API物件關係映射的整合層
  3. OXM - 提供抽象層
  4. JMS - 包含產出與使用訊息的功能
  5. Transaction - 支援programmatic與declarative交易管理

Web

  1. WebSocket - 支援two-way溝通(client-server)
  2. Servlet - 包含Spring MVC給予web應用的實作
  3. Web - 提供web導向整合的基本功能
  4. Portlet - 提供MVC實作，且鏡像web-servlet模組的功能

Misc.

  1. AOP
  2. Aspects - 提供AspectJ的整合
  3. Instrumentation -
    • 支援class instrumentation(在已編譯的.class檔案加入字節碼byte-code)
    • 以及class loader implementation(在執行時將java類別動態載入JVM)
  4. Messaging - 支援STOMP作為WebSocket的sub-protocol
    • STOMP: 流文本定向消息協議
  5. Test - 支援Spring組件(component)的測試
