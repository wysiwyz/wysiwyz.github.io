SAA-C03 AWS Solution Architect Associate 準備心得（上）

Motivation

自從去年八月考過 GCP PCA 之後，發現 (1) 想去的幾家公司使用的雲服務供應商都是用 AWS 居多，加上 (2) 某次聚會上遇到的進外商的優秀前同事有 AWS SAA-C03 證照，(3) 今年 AWS 新聞說要在台灣設立資料中心，以及 (4) 前公司提供了 AWS 考照半價補助但沒機會用到，還有 (5) 在 Facebook 社團看到 AWS 官方提供了兩種考照半價優惠的方案（一個要組隊報名，另一個只要在時限前完成官方部分課程）。

基於看到 Educate Program 再過一天就要報名截止的心態，我刻不容緩的完成兩章的指定課程，趕緊送出申請，並且在一週後的星期三收到報名成功通知信。

Requirement

報名成功通知信主要有三個重點：

  1. 要完成三個任務才能再填表單申請半價考試券 voucher

    • 完成 skills builder 某一個課程（要提供課程完成100%截圖）
    • 在 skills builder 某一個 exam set 考試達到 72 / 100 的成績

  2. 提供了一個 Notion URL 以及一個 Discord URL

    • 可以在 notion 裡面看到其他人的考試心得、考古題資源還有 AWS 學習影片

  3. 9/27 之前考試通過的話可以拿到 AWS 背包或者 AWS T-Shirt；分享心得可以拿到筆電支架

    許願 SAA 考過是拿到 T-shirt，但是 8/19 的考試券通知信是公告背包，也許可以拿來當地震逃難包的選手之一（？）

準備方向

  1. 學習面

    我主要是以 AWS Skills Builder 的影片為主，Udemy 或 Youtube 影片為輔。

    官方的 Skills Builder 使用下來自己體感覺得相較於 Google Cloud Skills Boost 使用起來，影片資源陳列的方式比較難一目瞭然。例如 GCP Skills Boost 打開可以直接看到有幾部影片、幾個 Quizzes，但 AWS Skills Builder 不一定每個章節都會有小考，這部分可能也是因為自己用的 AWS 是個人帳號，但是在使用 GCP Skills Boost 的時候用的是前公司帳號，有 Google Partner 加持。

  2. 實務面

    除了在公司有實際練習到幾個功能（少掉很多 Fancy a.k.a. 比較貴比較盤子的服務），另外也有看 FreeCodeCamp 一部有 50 多小時、由 ExamPro 製作的 SAA-C03 影片。

    自己在準備證照的過程中有知會公司，隱約覺得相較於前公司，主管對於考這種雲服務證照的支持態度有比較多保留態度。所以在這一個月的準備期間，我的首要原則是必須理論實務並重，然後最好要有一個自己的 AWS 帳號多練習把它練熟。

  3. 題目面

    • 使用 Educate Program 提供的 Notion 裡面的考古題

    • 使用 ExamTopics 考古題

    • 把 ExampTopics 考古題用 Google Apps Script 存進一張 Google Sheet

    • 用 Google Apps Script 將 Google Sheet 的題目，每隔 30 分鐘隨機抽一題題目/選項/Discussion，用 LineNotify 傳送到 Line ChatRoom

      有點像是利用單字卡加深印象

準備進度

7/30 - 7/31 完成 AWS Cloud Practitioner Essentials Chapter 01 ~ 02

7/31 - 8/7 完成 AWS Cloud Practioner Essentials Chapter 03 ~ 11; 看 Udemy AWS CLF-C02 影片

8/7 - 8/13 完成 AWS Skill Builder 以下兩個課程並提交 voucher 申請

  • Exam Prep Standard Course: AWS Certified Solutions Architect - Associate (SAA-C03)
  • Exam Prep Official Question Set: AWS Certified Cloud Practitioner (CLF-C02 - English)

8/19 - 9/20 看考古題 ＋ 補完官方 SAA-C03 的影片

  • ExamTopics 981 題（可能看不完但會拼命看完，看 user Discussion 很好玩）
  • 在 Google Apps Script 寫程式將 ExamTopics 題庫輸出到 Google Sheet，再每 30 分鐘抽一題目/選項/解析，以 Line Notify 送到自己的 Line Chatroom
  • 其它學習資源
    • 雲育鏈出版的 大話AWS雲端架構
    • FreeCodeCamp SAA-C03 50 hours Full Course with Demo 如果想要跟 QwikLab 一樣動手實作 (get your hand dirty) 但又不知道從何開始，推薦跟這個影片實作
    • Udemy 的課程（這兩個 CLF-C02、SAP-C02） SAP-C02 是 SAA-C03 再進階，是當初還沒看到 Educate Program 就買的課程，想說前主管說 GCP 不用考 associate 準備 professional 就好，那 AWS 應該也同理可證

先這樣，後續再補考試心得！
