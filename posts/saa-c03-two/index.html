準備心得

八月中旬自主模擬小考的期間，看完 ExamTopics 前 240 題之後，我發現 AWS Educate 暑期陪跑計畫的 Discord 群組有好心人分享使用關鍵字可以 Google Search 搜尋到剩下 241 ~ 981 題絕大多數的題目，所以我的做法是把這些一個個 copy 下來。聽起來很蠢，但因為它有些頁面都需要用 reCAPTCHA 之類的驗證碼確認是不是真人才能開啟，就沒有考慮研究網路爬蟲來節省時間了。

另外也有使用 Google Apps Script 將存下來的題庫 google docs 轉成 google sheet，加上 Line Notify 並且紀錄每題發送次數的小功能，這樣用餐、吃飯時間、睡前、上洗手間、等火車這些零碎的滑手機時間，都會有 line 小幫手每 30 分鐘丟給你一個題目以及精選 discussion，實在是太有助於時間不夠的考生了！（絕對不是因為line通知太煩所以提前測驗的）

測驗過程

原本因為 voucher 通知信上說考照成功獎需要在9/27前考照成功，才有資格申請，所以排 9/20 週五晚上考試。但後來 9/2 全部將 ExamTopics 的題目收進複習的小本之後，覺得一直刷題實在很沒趣，有更重要的 Keycloak 升級與 Kubernetes 實務一把火在燒，所以隔天就手刀上 PearsonVue 測驗登記網站提前測驗日期了。

提前測驗的行為現在想來真是一步險棋，因為我到 9/5 晚上六點半測驗前都還沒有全部重看完所有考古 981 題。加上台北 PearsonVue 考場實在有夠冷，坐在一間長寬約 100 cm 的單人間座位，看著考試介面上字體被拉寬 10~20% 的英文題目，總時長 140 分鐘一共 65 題。

測驗通知信說至少要在表定考試前 15 分鐘報到，怕下班時間塞車所以我提前 45 分鐘報到，為的是台北考場景觀很美的休息區（重點錯誤）。報到時要簽署兩份文件，櫃檯人員會核對雙證件以及再次確認考試科目代碼以及英文姓名，還需要像 costco 或者富邦辦卡那樣到白牆前拍照留紀錄。因為我報名時護照剛好快過期，致電詢問台北考場確定可以用身份證＋英文姓名正確的信用卡驗證身份。進考場前要將手機與行動裝置關機，個人物品寄放在櫃檯，監考人員會檢查你身上的口袋（上衣、褲子前後口袋）都有淨空，也需要將眼鏡拿下來放到平面上給監考官檢查，褲管也需要撩起來檢查。

考試過程中，值得一提的是，PearsonVue 的測驗題目介面也跟 Kyterion 一樣可以讓你 mark for review (小的不才，居然 mark 了 21 題，所以考完直覺自己小看了 PearsonVue 可能考爆了)，另外還可以變更佈景主題，預設是藍白底黑字，另外還有很多主題，像是 white on black、black on pink、white on blue（但我好奇是不是真的有人會更換，畢竟藍底白字看久了很容易視覺疲勞）。

考完需要填寫 14 題 survery，問題涵蓋考試題目範圍、考場環境。考完後不會直接顯示測驗結果，而是告訴你五個工作天內會收到測驗結果。我自己是當天晚上 8 點多考完，隔天早上醒來就看到測驗結果通知信（You just earned a badge from Amazon Web Services Training and Certification🎉 !）簡直不要太 Happy Friday。

應試心得

先說 AWS 的評分模型實在很神秘，看到 AWS Educate Ambassador 說會有一些不計分的題組，就是這些題目讓我走出考場覺得之前煞費苦心的準備，付出的 2400 台幣（150USD 扣掉半價 voucher）可能要付諸流水白忙一場了。

當然，參加測驗只是一種認識 AWS Cloud Provider 提供的服務，考題範圍很廣，但並沒有真的考到你怎麼實作以及實作要注意的眉角。公司的大佬說他手上有兩張 AWS 證照，後續還是持續看了十幾本書/看官網學習，在前公司的專案建置了一整套 AWS CodeDeploy 以及 AWS CodePipeline，另外也在公司上個專案實作了 AWS site-to-site VPN 的功能。

我自己也認同選擇題或多選題這類型的考試，講白點如果你很會考試、記憶力也不差夠熟悉考古題、也有基本的概念，用測驗評估實力確實是有一點放水，含金量不會比需要動手實作的 Kubernetes CKA 或 Elastic Certification 一樣，有需要動手 hands on 實作的題目。但也誠如其他分享考照心得的網上好人所述，確實不失為一種讓非本科轉職工程師建立自信心的方法之一。

最後感謝參與 AWS 陪跑計畫的大使與一起線上準備證照的大家，準備的過程才不會覺得孤獨，有一群人一起奮鬥的感覺真好。附上新鮮的測驗結果。
